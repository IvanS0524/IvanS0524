## 🚀 Proyectos Destacados

### 📦 [ProyectoPOO — Sistema de Gestión de Inventario](https://github.com/IvanS0524/ProyectoPOO)

> Proyecto académico desarrollado en la asignatura de Programación Orientada a Objetos.

Sistema completo de gestión para tiendas minoristas, construido en **Java con interfaz gráfica Swing**.

| Módulo | Descripción |
|--------|-------------|
| 🏷️ Productos | Registro, modificación, eliminación y consulta con precios y stock |
| 👥 Clientes | Gestión de clientes con datos de contacto |
| 🏭 Proveedores | Administración de proveedores de mercancía |
| 🛒 Transacciones | Compras (incrementan inventario) y ventas (decrementan inventario) con carrito |
| 💾 Persistencia | Almacenamiento y recuperación de datos en archivos binarios |
| 🖥️ Interfaz | GUI intuitiva con Java Swing |

**Tecnologías:** `Java` · `Swing` · `OOP` · `Archivos Binarios`
